bool instanceBackToBackEnable

Enable the instance back to back mode between PDB0 CH0 and PDB1 CH0 pre-triggers, forming a ring (PDB0_CH0_pretrigger7 -> PDB1_CH0_pretrigger0 and PDB1_CH0_pretrigger7 -> PDB0_CH0_pretrigger0). : this configuration is common between all PDB instances (configures the same register SIM_CHIPCTL[PDB_BB_SEL] even if configured for either PDB instance) Enabling when configuring PDB0, then disabling when configuring PDB1, will result in being disabled. : must not be enabled simultaneously with inter-channel back to back on any PDB instance.
